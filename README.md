还有这是课程作业。

如果你想要使用这份project，记得去godot官网搞个下载来（我使用的是godot.NET 4.7，虽说并没有任何使用到C#的部分），给自己的电脑装个rustup。

在设计上有参考https://github.com/godot-rust/demo-projects

隔壁的仓库有点老，但是他们最重要的思路是使用自定义节点+rust的形式替换了原生节点+.gd/.cs的传统模式，考虑到rust的性能，算得上更加安全的脚本。

有一些接口还是gd用起来方便些，所以你会看到gd+rust的语言组合。

不过我这个版本不会再考虑更新，也许有朝一日他们就超过我了呢。
